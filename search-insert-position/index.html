LeetCode 35 - search-insert-position

2021-02-03
LeetCode
  • Algorithms

搜索插入位置

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。你可以假设数组中无重复元素。

示例 1:

输入: [1,3,5,6], 5
输出: 2

示例 2:

输入: [1,3,5,6], 2
输出: 1

示例 3:

输入: [1,3,5,6], 7
输出: 4

示例 4:

输入: [1,3,5,6], 0
输出: 0

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/search-insert-position

暴力解法

class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        for i in nums:
            if i >= target:
                return i

时间复杂度：O(n)
空间复杂度：O(1)

二分法(写法一 )

class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        n = len(nums)
        left = 0
        right = n - 1   # target在[left,right]

        while (left <= right):
            middle = left + ((right - left) / 2)
            if nums[int(middle)] > target:
                right = int(middle) - 1 # target在左区间 [left,middle-1]
            elif nums[int(middle)] < target:
                left = int(middle) + 1  # target在右区间 [middle-1,right]
            else:
                return int(middle)

        return right + 1

时间复杂度：O(logn)
空间复杂度：O(1)

二分法(写法二 )

class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        n = len(nums)
        left = 0
        right = n   # target在[left,right)

        while (left < right):
            middle = left + ((right - left) / 2)
            if nums[int(middle)] > target:
                right = int(middle)     # target在左区间 [left,middle)
            elif nums[int(middle)] < target:
                left = int(middle) + 1  # target在右区间 （middle+1,right]
            else:
                return int(middle)

        return right + 1

时间复杂度：O(logn)
空间复杂度：O(1)
