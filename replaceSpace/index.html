替换空格

请实现一个函数，把字符串 s 中的每个空格替换成”%20”。

示例 1：

输入：s = “We are happy.”
输出：”We%20are%20happy.”

C

char* replaceSpace(char* s){
    int len = 0;        // 原始长度
    int count = 0;      // 空格个数
    while (s[len] != '\0') {
        if (s[len] == ' ') count++;
        len++;
    }
    int newLen = len + count * 2; //新长度
    s = (char*)realloc(s, newLen + 1);
    while (len >= 0 && newLen > len) {
        if (s[len] == ' ') {
            s[newLen--] = '0';
            s[newLen--] = '2';
            s[newLen--] = '%';
        } else {
            s[newLen--] = s[len];
        }
        len--;
    }
    return s;
}

时间复杂度：O(n)
空间复杂度：O(1)

Python3

运行时间：40 ms
内存消耗：14.9 MB

class Solution:
    def replaceSpace(self, s: str) -> str:
        return s.replace(' ', "%20")
}
